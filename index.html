3. Information We Collect

3.1 Provided by Schools

  • Student info: Name, class, roll number, admission year[cite: 1].
  • Parent info: Names and contact phone numbers[cite: 1].
  • Staff info: Name, title, class assigned, phone/email[cite: 1].

3.2 Provided by Users

  • Login credentials managed via Firebase Authentication[cite: 1].

3.3 Collected Automatically

We use Firebase for authentication tokens, session data, analytics, and crash reports to improve app performance[cite: 1].

4. How We Use Information

We use data solely to provide core functionality: attendance tracking, academic records, assignments, and communication between school and parents[cite: 1]. We do not use your data for advertising or commercial profiling[cite: 1].

5. Data Storage and Security

All data is stored securely on Google Firebase (Firestore) with industry-standard encryption[cite: 1]. Access is strictly enforced through role-based permissions and Firebase Security Rules[cite: 1].

6. Data Sharing

We do not sell or rent your information. We only share data with trusted services like Google Firebase required to operate the App[cite: 1].

7. Children's Privacy

Student accounts are created exclusively by administrators. We do not collect unnecessary data from minors and only use information for educational management features[cite: 1].
